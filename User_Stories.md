# User Stories

## Story 1
As a user, I want to see a list of requests so I can review incoming tasks.
Acceptance Criteria:
- Request list loads on app start
- Each request shows title and status

## Story 2
As a user, I want to accept a request so I can start working on it.
Acceptance Criteria:
- User can accept a request
- Request status updates to "accepted"

## Story 3
As a user, I want to view messages so I can communicate about a request.
Acceptance Criteria:
- Message list is displayed
- Messages are ordered correctly

## Story 4
As a user, I want to send a message so I can communicate with others.
Acceptance Criteria:
- User can enter message text
- Message appears in the list after sending
